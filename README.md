# AI-Scientists-FARS

This repository brings together generated papers, reviewer outputs, and supporting material for the FARS project.

The `AI-Scientist-generated-papers` folder is the main collection of papers produced by the different AI scientist systems used in this study.

Within that folder, `ICML-workshop-papers` contains the smaller paper set created during the earlier stage of the project, while `NeurIPS-paper-proposals` contains the larger full proposal set.

Folders such as `Cycle Researcher`, `Data-to-Paper`, `FARS`, `Sakana v1`, and `Sakana v2` correspond to the individual systems that produced those papers.

Folders ending in `_full` contain the extended full proposal versions for the corresponding systems.

The `Results` folder contains score files, summary tables, analysis outputs, and plotting artifacts derived from the generated papers.

The `AI-Reviewer-outputs` folder contains reviewer evaluations and related outputs.

The `FARS` folder contains project material associated with the FARS system itself.
